public subroutine set_radiative_cooling_values(rho_field, T_field, rc_field)

Uses

    • mod_interpolation
    • mod_types
    • mod_logging
    • mod_cooling_curves
    • mod_global_variables

Sets the radiative cooling attributes of the corresponding types. This is called after the equilibrium is initialised in the submodule.

Note

No cooling is applied when T0 is below the lower limit of the cooling curve. If T0 is above the upper limit of the cooling curve, pure Bremmstrahlung is assumed.

Warning

Throws an error if the cooling curve is unknown.

Type IntentOptional AttributesName
type(density_type), intent(in) :: rho_field

the type containing the temperature attributes

the type containing the radiative cooling attributes
