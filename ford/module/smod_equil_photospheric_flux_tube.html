photospheric_flux_tube_eq

This submodule defines a magnetic flux tube embedded in a uniform magnetic environment. In this case the flux tube is under photospheric conditions where the subscript e denotes the outer region. More specifically the equilibrium is defined as . The geometry can be overridden in the parfile, and is cylindrical by default for .

This equilibrium is taken from chapter 6, fig. 6.5 in Roberts, Bernard (2019). MHD Waves in the Solar Atmosphere. Cambridge University Press. DOI.

Note

For best results, it is recommended to enable mesh accumulation.

Note

Default values are given by

  • k2 = 0
  • k3 = 2
  • cte_rho0 = 1 : density value for the inner tube.
  • cte_p0 = 1 : pressure value for the inner tube.
  • r0 = 1 : radius of the inner tube.

and can all be changed in the parfile.
