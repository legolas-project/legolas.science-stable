Main handler for console print statements. The level of information printed to the console depends on the corresponding global variable called logging_level defined in the parfile.

Note

Values for logging_level can be set to

  • If logging_level = 0: only critical errors are printed, everything else is suppressed.
  • If logging_level = 1: only errors and warnings are printed.
  • If logging_level = 2: errors, warnings and info messages are printed. This is the default value.
  • If logging_level = 3+: prints all of the above, including debug messages.


Uses

    • mod_global_variables
    • mod_painting
